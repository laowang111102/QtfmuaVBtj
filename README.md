# 旅游景点分析系统 python588

## 项目概述


## 技术栈

- 后端：Flask
- 数据库：SQLite
- 前端：HTML, CSS, JavaScript

## 功能模块

- 登录注册
- 旅游数据总览
- 旅游景点词云分析
- 旅游景点信息管理
- 旅游景点评论可视化分析

## 系统角色

系统包含以下角色：

- 普通用户：可查看旅游信息，进行登录注册，发表评论等。
- 管理员：负责管理旅游景点信息，查看用户评论。

## 运行环境

- Python 3.8 或更高版本
- Flask 2.0 或更高版本
- SQLite 3.34 或更高版本

## 部署步骤

1. 安装Python环境
2. 安装依赖库（Flask, SQLite等）
3. 导入数据库文件
4. 运行Web应用程序
